# NEXUS — demo assets

Show, don't make them imagine. Everything here is self-contained — open the `.svg`
files in any browser; no build step.

| File | What it is |
|---|---|
| `90_SECOND_DEMO.md` | The timed narration for the live `decision-graph.html` aha. |
| `JUDGE_SCRIPT.md` | Exactly what to run and what each claim rests on (≤5 min). |
| `cascade.svg` | The aha as a rendered, **animated** asset — open in a browser to watch one broken belief drain four decisions and reprice `$40M`. |
| `cascade-result.png` | Static final-state still of the cascade (renders inline anywhere, incl. GitHub). |
| `recommendation-card.svg` / `.png` | The executive recommendation card: action · reason · `$24.1M` impact · confidence move · why-now · evidence · alternative. |

The interactive, data-driven versions live in `../nexus-landing/standalone/`
(`decision-graph.html` is the centerpiece). These SVGs are the at-a-glance previews for
the README and for a judge skimming before they run anything.
